# Contributing

Danke für dein Interesse! Kleine Hinweise:

- Bitte PRs auf branch `dev` anlegen.
- Code-Style: NASM sources: 4 spaces indent, lowercase directives.
- Tests: Wenn du Änderungen am Header machst, bitte `dumpbin /headers` und SHA256 der Ergebnis-EXE anhängen.
- Issues: beschreibe System (Windows Version), verwendete NASM Version, und ob du DOSBox / native Test verwendet hast.
